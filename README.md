# AIM PRACTICE on WEB!

## 内容

WebサイトでAIM練習ができるゲーム  
(PixiJSとGSAPの練習用にTSで作ったもの)

## 開発

### 環境

- [Node.js](https://nodejs.org/)
  - v12.\*
- [npm](https://www.npmjs.com/)
  - v7.\*

### ローカルで起動

```
git clone git@github.com:husky21st/AimLaaab.git
cd AimLaaab
npm i .
npm run dev
```

### ビルド

```
npm run build
```

## 主要ライブラリ

```
PixiJS
GSAP 3
```
